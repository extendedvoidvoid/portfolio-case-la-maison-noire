# Standard production — international video ad (France)

**Purpose:** map where our **stills animatic** sits in a real FR / international commercial pipeline — for entretien CR Films & Images.  
**Not legal advice.** Spec portfolio only.

---

## Chaîne classique (agence + prod)

```
ANNONCEUR
    ↓ brief / validation
AGENCE (création) — DA / CR / planning
    ↓ concept, script, storyboard, animatic
    ↓ (souvent) pitch concurrence
PRODUCTION (maison de prod publicitaire)
    ↓ devis, équipe, tournage, post
LIVRABLES multi-formats (TV / digital / social / international cuts)
```

Sources types: fiches métier producteur exécutif (ESEC, Cinecreatis), phases film pub (Medium Capucine Reyt), coûts tournage (Wakup).

---

## Phases standard

| Phase | Qui | Livrables typiques | Où on est |
|-------|-----|-------------------|-----------|
| **1. Brief** | Annonceur → agence | Objectifs, cible, budget, territoires, timing | Offre poste = ce contexte |
| **2. Conception** | CR + DA + planning | Insight, promesse, script, tone of voice | **Script *Tu me connais*** |
| **3. Storyboard / frames** | CR / DA / illustrateur | Plan par plan, timing | **Shots 01–10 installés** |
| **4. Animatic** | Prod / monteur / CR | Montage stills (ou rough) + temp sound | **`animatic_v1_stills.mp4` ← NOW** |
| **5. Validation client** | Annonceur + agence | Go / notes / lock | **À faire par toi** |
| **6. Préparation** | Producteur exécutif, 1er AD, repérages | Devis, planning, casting, lieux, autorisations | Après lock créatif |
| **7. Tournage** | Réalisateur + équipe | Rushes (souvent 1–3 jours pour un film pub) | Pas encore |
| **8. Post-production** | Monteur, étalonneur, son, VFX, musique | Offline → online → grade → mix → masters | Plus tard: slow-motion pass |
| **9. Livraison** | Prod + agence | Masters 16:9, 9:16, 4:5, clean / subtitled, international | 360 déclinaisons |
| **10. Diffusion** | Média / digital | TV, BVOD, social, site | Hors scope portfolio |

## Rôles clés (France, pub)

| Rôle | Fonction |
|------|----------|
| **Concepteur-rédacteur** | Concept, script, ton, collaboration DA — *ton poste cible* |
| **Directeur artistique** | Image, direction visuelle, cohérence |
| **Directeur de création** | Validation créative agence |
| **Producteur (prod)** | Porte le film côté société de production |
| **Producteur exécutif** | Budget, délais, exécution, équipe du prep au wrap |
| **Réalisateur** | Mise en scène tournage (souvent briefé via boards + animatic) |
| **Chargé / directeur de production** | Logistique, intermittents, planning |

## International specifics

- **Versioning:** master + cuts durée (15″ / 30″ / 60″) + **formats digitaux** (9:16, 1:1).  
- **Adaptation:** sous-titres / versions sans texte / voix-off local (notre film: **no VO** = plus simple à exporter).  
- **Tournage FR vs foreign:** pub FR souvent sous conventions collectives production ; beaucoup de films “internationaux” tournent hors France (coût) puis post en FR ou London.  
- **Crédits d’impôt:** le cinéma/audiovisuel fiction a des mécanismes ; la **pub pure** est souvent moins aidée (notes sectorielles) — à ne pas inventer en entretien.

## Où l’animatic se place (important pour l’entretien)

L’**animatic stills** (ce que nous livrons maintenant) est le standard **avant** engagement budget tournage :

1. Valide **rythme + narration** sans rush.  
2. Outil de discussion **agence ↔ annonceur ↔ réalisateur**.  
3. Base pour devis (nombre de setups, cast, lieux).  
4. Ensuite seulement: motion pass / tournage réel.

**Notre file:** `assets/video/animatic_v1_stills.mp4` (~70s, hard cuts, no motion).
