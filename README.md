# La Maison Noire

**Spec fragrance film** — portfolio case study for a Chanel Paris candidature (Parfums Beauté / films & images).

Codes only. No trademark. No real-person likeness. Unlabeled bottle.

**Live:** https://extendedvoidvoid.github.io/portfolio-case-la-maison-noire/

---

## What this is

A 75-second European luxury fragrance film concept, presented as a director’s storyboard + still-frame animatic:

| Pillar | Intention |
|--------|-----------|
| **Newton** | Power through observation; curved body as authority |
| **Lagerfeld** | Tweed, pearl, glove — self as garment |
| **Helleu** | Scent before name; simplicity as radical luxury |

**Version 001 — Equilibrium** (default). Shared character: **MADAME_X**.
